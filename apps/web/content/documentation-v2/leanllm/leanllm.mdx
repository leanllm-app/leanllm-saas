LeanLLM is a Python SDK that wraps provider calls via [LiteLLM](https://github.com/BerriAI/litellm) and emits one structured `LLMEvent` for every request.

You get prompt/response, parameters, tokens, cost, latency, lineage, and labels with async persistence in the background, so your request path stays fast.

## Why teams use LeanLLM

- **Visibility by default:** understand cost per feature, per user, and per request.
- **Zero lock-in:** works with any provider supported by LiteLLM.
- **Replay confidence:** reproduce any captured event with deterministic replay.
- **Built for production:** async worker architecture that does not block responses.
