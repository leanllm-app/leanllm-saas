LeanLLM is a Python SDK that wraps any provider call routed through [LiteLLM](https://github.com/BerriAI/litellm) and emits a structured `LLMEvent` per call: prompt, response, parameters, tokens, cost, latency, lineage, labels. Events are persisted asynchronously through a worker thread, so the request path is never blocked.

This section covers how to install the SDK, configure persistence, and use each feature.
